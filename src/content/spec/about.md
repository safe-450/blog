
本网站使用 **Astro** 框架构建，采用 [Mizuki](https://github.com/LyraVoid/Mizuki) 主题。

::github{repo="LyraVoid/Mizuki"}

## 🌟 主题特性

### 🎨 设计与用户体验
- **现代优雅设计** - 简洁的界面，精美的排版
- **完全响应式** - 适配从移动端到桌面端的所有设备
- **深色/浅色模式** - 自动主题切换，过渡平滑
- **精美排版** - 使用 JetBrains Mono 字体提升可读性
- **流畅动画** - 页面过渡和交互元素流畅自然

### 🔍 内容与搜索
- **高级搜索** - 由 [Pagefind](https://pagefind.app/) 驱动，快速精准的搜索结果
- **增强 Markdown** - 扩展语法，支持代码高亮和数学公式
- **交互式目录** - 长文章自动滚动导航
- **RSS 订阅源生成** - 自动生成订阅源，随时获取更新
- **阅读时间估算** - 了解文章阅读所需时间
- **文章分类** - 使用标签和分类组织内容

### 📱 特色页面
- **追番页面** - 追踪动漫观看进度，支持评分
- **友链页面** - 精美卡片展示友站链接
- **日记/动态页面** - 像社交媒体一样分享生活瞬间
- **归档页面** - 时间线视图展示所有文章
- **关于页面** - 可自定义的个人介绍（就是本页面！）

### 🛠 技术特性
- **增强代码块** - 由 [Expressive Code](https://expressive-code.com/) 驱动
- **数学公式支持** - 使用 KaTeX 渲染 LaTeX 数学表达式
- **图片优化** - PhotoSwipe 画廊配合懒加载
- **SEO 优化** - 内置站点地图和 meta 标签，提升搜索引擎可见性
- **性能优化** - 缓存和优化策略实现快速加载
- **评论系统支持** - 可集成 Twikoo 评论系统

### 🎯 高级 Markdown 功能
- **提示框与警告框** - 使用 `> [!NOTE]`、`> [!TIP]`、`> [!WARNING]` 创建精美提示框
- **数学公式** - 使用 `$行内公式$` 和 `$$块级公式$$` 语法编写 LaTeX 数学公式
- **GitHub 卡片** - 使用 `::github{repo="用户/仓库"}` 嵌入仓库卡片
- **语法高亮** - 高级代码高亮，支持行号显示
- **复制代码按钮** - 一键复制代码功能
